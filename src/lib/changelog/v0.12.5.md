# Scanopy v0.12.5 Release Notes

## New Features

- Add topology sharing capabilities, allowing users to share network maps via direct link or embed code for easier collaboration and presentation ([#95](https://github.com/scanopy/scanopy/pull/95)).
- Introduce demo mode, letting users explore Scanopy features with a simulated environment hosted at demo.scanopy.net ([#102](https://github.com/scanopy/scanopy/pull/102)).

## Improvements

- Improve subnet visualization by optimizing horizontal compression and preventing overlap, resulting in clearer network maps ([#107](https://github.com/scanopy/scanopy/pull/107)).
- Enhance topology settings by ensuring correct persistence and clearer edge labeling for easier network interpretation ([#108](https://github.com/scanopy/scanopy/pull/108)).
